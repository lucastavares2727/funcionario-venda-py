Ola, este codigo sera responsavel por fazer a comunicação com o usuario quando um vendedor
alcaçar a meta de 50mil em suas vendas mensais, a comunicação sera feita atravez de um sms que iremos
pedir parar o codigo enviar para um determinado número que estara no codigo fonte.

1 - Primeiro devemos exportar as bibliotecas necessarias para esse codigo funcionar corretamente.
1-1 Pandas, responsavel pela integração do python e Excel
1-2 Twilio responsavel por mandar o sms para o celular.
1-3 Openpyxl responsavel pela integração do python e Excel.
1-4 Vale ressaltar que é necessario a criação de uma conta no site da twiliio "https://www.twilio.com/docs/libraries/python" para inserir no codigo fonte onde estar especificado.
